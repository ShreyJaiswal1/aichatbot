# Honey AI Chatbot

This chatbot replies in hinglish, made using groq api. Further features will be upcoming soon...

# Steps to localhost
1. Go to the backend folder
```
cd backend
```
2. Install Dependencies
```
npm install
```
2. Run using
```
node server.js
```
4. Open the html and Enjoy
